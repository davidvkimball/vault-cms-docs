# Vault CMS Documentation

This repository contains the source code for the official [Vault CMS documentation](https://docs.vaultcms.org).

## About Vault CMS

Vault CMS allows you to use [Obsidian](https://obsidian.md) as a powerful, local-first content management system for your [Astro](https://astro.build) website. It features auto-detection of content types, frontmatter analysis, and a preconfigured workspace optimized for Astro workflows.

## Development

This is an Astro project using the [Starlight](https://starlight.astro.build) theme.

### Commands

| Command | Action |
| :--- | :--- |
| `pnpm install` | Install dependencies |
| `pnpm dev` | Start local development server |
| `pnpm build` | Build for production |
| `pnpm preview` | Preview production build locally |
